## 一、推理与前处理层（最快见效，半天内可调完）


| #   | 优化项              | 建议方案                     | 当前实施状态  | 备注                                                 |
| --- | ---------------- | ------------------------ | ------- | -------------------------------------------------- |
| 1   | **VAD 必须做干净**    | Silero VAD 或 webrtcvad   | ⚠️ 部分实现 | 自定义 RMS-based VAD，非标准方案；实验建议 `enable_vad=False`    |
| 2   | **注册时多段平均**      | 3-5 段，每段 ≥3 秒，L2 归一化后取均值 | ✅ 已实现   | 支持多段 enrollment + 均值化，可选 SNR 加权                    |
| 3   | **验证段长度控制**      | 3-10 秒，长音频切块平均           | ✅ 已实现   | `verify_crop_mode` + `verify_buffer_keep_secs` 可配置 |
| 4   | **余弦相似度 + 阈值校准** | DET 曲线找 EER 阈值           | ⚠️ 部分实现 | 使用余弦相似度，但阈值 0.55 是手动校准，非自动 DET                     |


**实施细节：**

- VAD: `src/wespeaker/wespeaker.py` lines 203-249，自定义 RMS 能量阈值
- 多段平均: `src/wespeaker/wespeaker.py` lines 314-336，支持 SNR 加权
- 长度控制: `src/wespeaker/wespeaker.py` lines 278-280, 370-390
- 阈值: 0.55 基于实验数据校准（clean 通过率 98.3%）

## 预期效果（专家评估）

### 5-6 人闭集场景（家庭用户）

- **理想条件**（近场、安静、配合度好）：98-99.5%
- **现实条件**（1-3 米、偶有背景音）：95-98%
- **困难条件**（远场、强噪、短语音）：85-93%

### 性能指标目标


| 指标             | 目标值     |
| -------------- | ------- |
| 识别准确率（>3 秒语音）  | ≥ 97%   |
| 陌生人拒识率（FAR）    | ≤ 2%    |
| 家人识别率（FRR）     | ≤ 3%    |
| 端到端延迟（RK3588S） | ≤ 200ms |


### 为什么 5-6 人场景友好

- 类间距离大、决策边界宽松
- 只需找 top-1 最近邻，不需要严格阈值
- 注册数据完全可控（用户配合度高）
